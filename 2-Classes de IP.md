
  ## Classes de IP

- Classe A - 1º número entre 0 e 127;

- Classe B - 1º número entre 128 e 191;

 - Classe C - 1º número entre 182 e 223

 - Classe D - 1º número entre 224 e 239

- Classe E - 1º número entre 240 e 255

>Classes A, B e C são usadas pela internet, todo IP de classe A cabe 16.777.216 endereços de IPS. A classe B cabe 65.536 endereços de IP. E a classe C cabe 256 endereços de IPS

>Classes D e E são redes especiais para situações específicas

---

 `Unicast` = mensagem para uma única pessoa

`Multcast` = mensagem para várias pessoas

`Broadcast` = mensagem para todas as pessoas da mesma rede

`Anycast` = mensagem para qualquer um que tiver perto
